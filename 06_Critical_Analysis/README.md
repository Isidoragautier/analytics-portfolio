# Critical Analysis - Australian Vehicle Prices

**Tools:** Python, Pandas, Matplotlib, Seaborn
**Dataset:** Australian vehicle price listings
**Course:** Macquarie University

## What this project covers
Critical analysis of Australian vehicle pricing data including data exploration,
statistical analysis, and insight generation on pricing trends across vehicle
types, brands, and conditions.

## Skills demonstrated
Data analysis, statistical reasoning, critical thinking, data visualisation, insight communication

## Files
- `critical_analysis.ipynb` - full analysis notebook
- `data/australian_vehicle_prices.csv` - dataset
