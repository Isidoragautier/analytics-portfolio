# Vehicle Price Analysis

**Tools:** Python, Pandas, Matplotlib, Seaborn, Scikit-learn
**Dataset:** Car sales data
**Course:** Macquarie University portfolio project

## What this project covers
Two-part analysis of vehicle pricing data.

### Part 1 - Exploratory Data Analysis
- Data cleaning and preprocessing of car sales dataset
- Distribution analysis of price, mileage, age, and brand
- Correlation analysis between price and key features
- Visualisations to identify pricing patterns

### Part 2 - Predictive Modelling
- Built regression models to predict car sale price
- Feature engineering and selection
- Model evaluation using RMSE and R-squared

## Skills demonstrated
Data cleaning, EDA, regression modelling, feature engineering, data visualisation, model evaluation

## Files
- `01_Car_Price_EDA.ipynb` - exploratory data analysis
- `02_Car_Price_Modelling.ipynb` - predictive modelling
- `data/car_sell.csv` - raw dataset
- `data/car_sells_clean_data.csv` - cleaned dataset
