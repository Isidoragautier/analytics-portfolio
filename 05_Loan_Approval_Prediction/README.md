# Loan Approval Prediction

**Tools:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn
**Dataset:** Loan approval dataset
**Course:** Macquarie University portfolio project

## What this project covers
Classification analysis to predict whether a loan application will be approved
based on applicant attributes.

## What I did
- Cleaned and preprocessed loan application data
- Performed exploratory analysis of approval rates by income, credit history,
  loan amount, and applicant profile
- Built and evaluated classification models
- Assessed model performance using accuracy, precision, recall, and confusion matrix

## Skills demonstrated
Classification modelling, data preprocessing, EDA, model evaluation, binary prediction

- `Loan_Approval_Classification.ipynb` - full analysis notebook
- `data/loan_approval.csv` - dataset
